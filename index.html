AI & Agentforce Hub

Enterprise AI Standards & ChatPwC Guidelines

Before deploying generative models or utilizing internal LLMs like ChatPwC, you must adhere strictly to the PH GTM data masking mandates. Client data, PII, and proprietary architecture MUST be scrubbed before hitting any AI gateway.

ChatPwC Approved Uses

  • Generating Apex test class scaffolding
  • Explaining complex code blocks or LWC concepts
  • Drafting technical documentation (no client names)
  • Regex pattern creation and validation

Strictly Prohibited

  • Pasting raw client data or unmasked SOQL results
  • Uploading proprietary client architecture diagrams
  • Using non-approved public LLMs (e.g., ChatGPT)

All Prompt Builder templates passing object data MUST be approved by the Security Architecture board prior to UAT deployment.
